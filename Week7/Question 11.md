# 输入只读图灵机等价于DFA

## 证明：输入只读图灵机等价于DFA。

DFA显然属于只读图灵机。证明反面即可。

对于语言来说，其输入长度必然有界，不妨设之为$n$。对于任何长度为$k$的输入只读图灵机识别的输入，我们构造如下的DFA识别它：

设图灵机状态集$Q$，构造$Q' = Q \times \{0, 1, \ldots, k\}$，对于任何输入带上的转移$\delta(q, a) = (q, a, L)$，DFA转移为$\delta((q, i), a) = (q', i-1)$; 同理，$\delta(q, a) = (q, a, R)$，DFA转移为$\delta((q, i), a) = (q', i+1)$。对于任何输入外的转移，我们有$\delta((q, 0), a) = (q', 0)$。任何两个带子之间的转移为$\delta(q, a) = (q', b, L), \delta(q, a) = (q, a, R)$，对应$\delta((q, 0), a) = (q', n), \delta((q, n)) = (q, 0)$。

图灵机在任何操作后接受，DFA都接受，因此二者等价。

我们构造DFA:
$$
\bigcup_{i=0}^n L(D_i) = L(TM)
$$
其中$D_i$为识别长度为$i$的只读图灵机的语言的DFA，注意到DFA的语言对于并运算封闭，因此这也是个DFA的语言。证毕。
